# Ejercicios de Bases de Datos

Este repositorio contiene ejercicios de bases de datos realizados durante mi formación como estudiante del programa Técnico en Programación de Software del SENA.

## Tecnologías utilizadas

- SQL
- MySQL
- MySQL Workbench

## Contenido del repositorio

Los ejercicios incluyen:

- Creación de bases de datos
- Creación de tablas
- Inserción de datos
- Consultas básicas
- Uso de JOIN
- Consultas para obtener información específica

## Archivos del proyecto

- Bd_clinica_actividad_4.sql → Creación de la base de datos
- Datos_clinica_actividad_4.sql → Inserción de datos
- Consultas_basicas_clinica_actividad_4.sql → Consultas básicas
- JOIN_clinica_actividad_4.sql → Consultas con JOIN

## Objetivo del proyecto

Practicar el diseño y manejo de bases de datos utilizando SQL, aplicando consultas para manipular y obtener información de manera eficiente.
